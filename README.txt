DiskActivityLed
===============

Lightweight software HDD/SSD activity indicator for Windows.

DiskActivityLed is a small Windows utility that shows disk read/write activity as a visible on-screen or tray-style indicator. It is useful on laptops, mini PCs, compact desktop cases, remote desktop sessions, virtual machines, servers, and other systems where a physical disk activity LED is missing, hidden, or not visible.

The main purpose of the program is simple: to help you understand whether the computer is actually frozen, or whether it is still actively reading or writing data.


Key features
------------

- Visual disk activity indication for HDD, SSD and NVMe drives.
- Separate read/write activity display.
- Useful when working through Remote Desktop / RDP.
- Useful on computers without a physical HDD LED.
- Lightweight and energy-conscious design.
- Customizable appearance: position, colors, font, labels and indicator style.
- Optional start with Windows.
- Portable usage is possible, depending on the release package.
- No advertising.
- No telemetry.
- No background Internet activity is required for normal operation.


Typical use cases
-----------------

- You work on a laptop or mini PC without a disk activity LED.
- You connect to a computer through Remote Desktop and cannot see the physical case indicators.
- A program or installer stops responding visually, and you want to know whether the system is still busy.
- You want a small always-visible activity indicator instead of opening Task Manager or Resource Monitor.
- You prefer simple, old-school system feedback without a heavy monitoring dashboard.


System requirements
-------------------

- Microsoft Windows 10 / 11
- 32-bit or 64-bit Windows
- Standard user account is usually enough for normal use

Note: exact compatibility may depend on the specific release version.


Installation
------------

Portable version:

1. Download the portable ZIP archive.
2. Extract it to a folder of your choice.
3. Run DiskActivityLed.exe.
4. Do not run it directly from inside the ZIP archive.


Safety and privacy
------------------

DiskActivityLed is intended to be a small local Windows utility.

The program does not need an Internet connection for its normal work. It is not intended to collect personal data, send telemetry, show advertisements, or connect to remote servers.

DiskActivityLed
===============

Лёгкий программный индикатор активности HDD/SSD/NVMe для Windows.

DiskActivityLed — небольшая утилита для Windows, которая показывает активность чтения и записи дисков в виде наглядного экранного или tray-индикатора. Она полезна на ноутбуках, мини-ПК, компактных корпусах, удалённых рабочих столах, виртуальных машинах, серверах и других системах, где физического индикатора активности диска нет, он скрыт или его не видно.

Главная задача программы простая: помочь понять, компьютер действительно завис или всё ещё активно читает/записывает данные.


Основные возможности
--------------------

- Визуальная индикация активности HDD, SSD и NVMe-дисков.
- Отдельное отображение чтения и записи.
- Удобно при работе через Remote Desktop / RDP.
- Удобно на компьютерах без физического HDD LED.
- Лёгкая и энергоэффективная работа.
- Настраиваемый внешний вид: положение, цвета, шрифт, подписи и стиль индикаторов.
- Опциональный запуск вместе с Windows.
- Возможность portable-использования, если такой пакет есть в релизе.
- Без рекламы.
- Без телеметрии.
- Для обычной работы не требуется подключение к Интернету.


Типичные сценарии использования
-------------------------------

- На ноутбуке или мини-ПК нет индикатора активности диска.
- Вы работаете через удалённый рабочий стол и не видите индикаторы на корпусе компьютера.
- Программа, установщик или сама Windows визуально «задумались», и нужно понять, идёт ли активная работа с диском.
- Хочется иметь маленький постоянный индикатор, не открывая Диспетчер задач или Монитор ресурсов.
- Нужна простая системная подсказка без тяжёлых мониторинговых панелей.




Системные требования
--------------------

- Microsoft Windows 10 / 11
- 32-битная или 64-битная Windows
- Обычно для работы достаточно обычной учётной записи пользователя

Примечание: точная совместимость может зависеть от конкретной версии релиза.
